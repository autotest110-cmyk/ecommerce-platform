 #E-Commerce Platform (Demo)
A full-stack e-commerce demo application built to showcase modern web development practices.

## Features
- Product listing & details
- Shopping cart
- User authentication
- Admin dashboard
- Order management

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT
